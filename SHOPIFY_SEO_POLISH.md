# Shopify SEO Polish Pack

Store audited: `https://refined-aesthetics-9619.myshopify.com/`
Date: 2026-03-24

## Executive summary

Main issues found:
- Store is still on the `myshopify.com` domain instead of a branded primary domain.
- Homepage SEO is extremely thin.
- GA4 is not visibly installed on the storefront HTML.
- Product pages are indexable but several descriptions feel imported rather than optimised for search + conversion.
- Collection and page content can do more of the commercial SEO heavy lifting.
- Internal linking between clinic trust content and product pages should be stronger.

## Highest-priority actions

1. Connect a branded primary domain
   - Best option: `shop.refinedmedicalaesthetics.uk`
   - Set as Shopify **primary domain**
   - Ensure canonicals switch away from `myshopify.com`

2. Install / verify GA4
   - Track at minimum:
     - `view_item`
     - `add_to_cart`
     - `begin_checkout`
     - `purchase`
   - Confirm events in GA4 DebugView

3. Add store domain to Google Search Console
   - Submit sitemap: `/sitemap.xml`
   - Verify indexed pages, exclusions, and performance queries

4. Replace weak homepage metadata
5. Upgrade collection copy
6. Optimise top product metadata + copy
7. Add FAQ content to collection and key product pages

3. How to use
4. Who it is for
5. Ingredients
6. Nurse-led trust note
